# Counselling – Interactive Infographic

An interactive educational infographic developed as an academic project for the Counselling – Educational Guidance course.

## Features

- Modern scrollytelling experience
- Responsive design
- Smooth animations
- Interactive educational activities
- QR learning resources
- HTML, CSS and JavaScript only
- GitHub Pages ready

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)

## Purpose

This project aims to communicate the principles of Counselling through an engaging and interactive web experience rather than a traditional static infographic.
